# Code

This repository contains some functions and scripts I developed during my research,  
mainly for analyzing and visualizing astrophysical simulations.  

---

## Research Context
These tools were used in my Master's thesis project at the **University of Potsdam**,  
where I study **AGN feedback** and **intracluster medium dynamics** using **AREPO** and **IllustrisTNG** simulations.
